How to run project

For running web server just you need to run run_me.py file. This module instantiate our WebServer, then registers defined actions on it and then start it.

Different Modules

engine module

This module is the core of our webserver and has some features like starting webserver and registering action on it.

actions

for adding any new functionality to our website we need to create an Action and registering it on our webserver instant. Every Action should have:

  • A regex what present urls that this action will call for them.
  • A mime_type present the mime_type of response
  • A response which creates response correspond to request parameters

you can see different implementations of Action in actions.py module
